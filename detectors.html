Supported Detectors

Detectors in the Github areaDetector Project

Beginning with release R2-0 (March 2014) the areaDetector module is in the areaDetector project on Github. This project is organized as a top level module and a set of submodules, e.g. ADCore, ADEiger, ADGenICam, etc.

The following table provides links to the github repository, the documentation, release notes, name of maintainer, and repository status.
